artifactory
===========

Python Library to the Artifactory API

#Configuration:

artifactory/artifactory/config/user.cfg.example

```
[artifactory]
hostname = YOUR_ARTIFACTORY_URL
user = YOUR_ARTIFACTORY_USERNAME
password = YOUR_ARTIFACTORY_PASSWORD
```
